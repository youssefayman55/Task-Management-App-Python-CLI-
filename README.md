# Task-Management-App-Python-CLI-
A simple command-line Python project that helps users create, manage, update, and delete daily tasks interactively.

🚀 Project Description

This project allows users to:

1- Create a list of tasks

2- Add new tasks

3- Update existing tasks

4- Delete tasks

5- View all tasks

It is a beginner-friendly project to practice:

1- Lists

2- Loops

3- Conditions

4- User input handling

🛠️ Tech Stack:

Python 🐍 (No external libraries required)

▶️ How to Run:

1- Save the script as main.py

2- Run it using:
   python main.py

💻 How It Works:

1. Start the App

The program starts and asks how many tasks you want to add initially.

3. Store Tasks

Tasks are stored in a Python list.

5. Menu Operations
6. 
The user can perform actions using a menu:

1- ➕ Add Task

2- ✏️ Update Task

3- ❌ Delete Task

5- 🚪 Exit Program

🎯 Features:

2- ✏️ Update tasks easily

3- ❌ Delete tasks

4- 👀 View all tasks
